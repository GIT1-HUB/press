# 此项目作用

vue-dynamic-form-element-master (opens new window)是一个基于vue和element-ui的可json配置表单组件，它的作用是实现json管理表单，减少重复代码的编写

# 下载

演示项目地址：dynamic-form (opens new window) 组件在components下的dynamic-form文件

# 使用

  1. 引入dynamic-form
	import DynamicForm from '@/components/dynamic-form'
  1. 注册组件
	components: {
  	    DynamicForm
	},
  1. 在template中使用
	<dynamic-form ref="form" v-model="formData" :model="formModel" />
	<el-button type="primary" @click="submit">提交</el-button>
  1. 在data中定义formData以及formModel
